## 수정된 회로 설계 핵심 사항

### 전원부 수정 회로

```
배터리(+) ─── [R1: 100kΩ] ─── GATE ─── [BZT52C12: 12V] ─── GND
                               │
배터리(+) ─── DRAIN(DMP2305U) ─── SOURCE ─── AP63205WU-7 VIN
                                              (4.2V~32V OK)
                                                 │
                                              5V 출력
                                                 │
                                    ┌────────────┤
                                    │            │
                               [AMS1117-3.3]  [ADP3338-3.3]
                               (디지털 3.3V)  (아날로그 3.3V)
                                    │
                               SOT-223 탭 → GND 동박 ≥1cm²
```

### STM32H743 전원 필터 수정

```
3.3V ─── [BLM18PG601: 600Ω@100MHz] ─── VDDA (Pin 11)
                                          ├── 10μF (X5R)
                                          ├── 1μF
                                          └── 100nF

3.3V ─────────────── VREF+ (Pin 21)
                      ├── 1μF
                      └── 100nF

GND ─── VCAP_1 (Pin 33) ─── 2.2μF (X5R, 6.3V)
GND ─── VCAP_2 (Pin 57) ─── 2.2μF (X5R, 6.3V)   ← 기존 누락!
```

### 로드셀 신호 체인 수정

```
로드셀 (5V 여기)
    │
  [R: 10kΩ] ─── IN+ ─── INA128UA ─── OUT ─── [R:1kΩ] ─── AINxP
  [BAV99]        │      (VDD=5V)               [C:150nF]
                IN- ─── VREF=1.65V             [C to GND]
                        │                          │
                [BAV99] │               AINxN ─── 1.65V (VDD/2)
                        │
                GAIN 설정: RG = 100kΩ/(G-1)
                G = 1650mV / (로드셀_full_scale_mV)
                예) 5V × 2mV/V = 10mV → G = 165 → RG = 613Ω
```

> **주의**: INA128은 5V 단전원에서 VDD/2 = 2.5V 공통모드 입력 지원. 허용 VCM = -4V ~ Vcc-1.25V = 3.75V → 2.5V OK.

### ADS1234 전원 수정

```
시스템 5V ─── AVDD (Pin) ─── 10μF + 100nF (기존 AGND 레일에서 공급)
3.3V      ─── DVDD (Pin) ─── 100nF
AVDD > DVDD 순서 또는 동시 인가 (역순 래치업 주의)
```

### CAN STBY 핀 수정

```
TJA1051 STBY ─── GND (직결)  ← 항상 Normal mode
             또는
TJA1051 STBY ─── [10kΩ] ─── GND (풀다운 + GPIO 제어 가능)
```

## 확인된 OK 항목

- TJA1051T/3J 3.3V 로직 호환: VCCIO=3.3V 인가 시 OK
- CAN-FD Classic CAN 하위 호환: FDCAN_FRAME_CLASSIC 설정으로 OK
- FDCAN1/2 핀 할당 (PD0/1, PB12/13): STM32H743 AF 매핑 OK
- INA219 배터리 모니터 버스 전압: 25.2V < 26V 측정 한계 OK
- ADP3338 아날로그 LDO: 노이즈 40μVrms, 열 여유 OK
- ADS1234 데이터 레이트: PDIV=00으로 채널당 1kHz OK
- REF3325 출력 전류: 10mA >> ADC 요구 전류 OK
- SWD 핀 (PA13/PA14): 표준 SWD OK
- STM32H743 SMPS: H743에는 없음 → 외부 SMPS 부품 불필요 (단순화)
- JST-GH 4핀 CAN 커넥터: 전류/전압 정격 OK
