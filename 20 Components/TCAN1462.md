# TCAN1462 — REJECTED

> ❌ **내 초기 선택 → 철회.** [[ISO1050]] 로 대체.
> 대체 이유: [[GND Bounce Protection]] · [[RECONCILIATION]]

## 왜 채택 안 했나

비격리 트랜시버 — exosuit 의 모터 스위칭 환경 (GND bounce 2.5V) 에서 TXD/RXD 핀 절대최대 (±0.3V) 초과 → **이전 보드 반복 파손 원인**.

## 참고용 사양 (미사용)

| 항목 | 값 |
|---|---|
| CAN-FD | ✅ 최대 5 Mbps |
| 격리 | ❌ |
| VIO | 3.3V / 5V |
| VCC | 5V |
| 가격 | ~$2 |

## 왜 원래 선택했었나 (반성)

- CAN-FD 5Mbps 지원 (미래 대비)
- 저렴
- 소형 SO-8

→ 이 모든 장점을 **신뢰성이 이겼음**. exosuit 환경에선 격리가 필수.

## CAN-FD 원하면

ISO1050 은 클래식 CAN 1Mbps 만. CAN-FD 격리 필요하면:
- **ISO1042** (TI, 격리 CAN-FD 2~5 Mbps) — 핀 호환, 업그레이드 경로
