# MGJ2D05050SC

## 한 줄 요약

5V → 5V 2W 격리 DC/DC 컨버터. [[ISO1050]] VCC2 (버스 측) 독립 전원 공급.

## 사양

- 입력: 4.5~5.5V (시스템 5V 레일)
- 출력: 5V 400mA (2W)
- 격리 내압: 5200 Vrms
- 효율: ~85%
- 패키지: SIP-7 (through-hole)
- 동작 온도: -40~105°C
- 가격: ~$10

## 왜 필요한가

[[Isolated CAN]] 아키텍처에서 ISO1050 의 VCC2 (버스 측) 는 MCU 측 5V 와 **전기적으로 분리** 돼야 함. 일반 5V 레일을 공유하면 격리 의미 사라짐.

MGJ2D05050SC 는 **5V 입력을 격리된 5V 출력** 으로 변환 → VCC2 공급.

## 외부 회로

```
시스템 5V (MCU 측 AGND) ─── Vin (Pin 1)
                         ─── GND (Pin 2)

                    ═══ 격리 경계 ═══

격리 5V (PGND)          ─── Vout (Pin 5)
GND2 (PGND)            ─── GND (Pin 4)
                        Pin 3, 6, 7 = NC 또는 trim

Vout ─[10µF + 1µF + 100nF]─ GND2
     └─[BZT52C5V6]────────  GND2  (5.6V 클램프)
     └─[ISO1050 VCC2]
```
