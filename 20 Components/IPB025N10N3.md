# IPB025N10N3

## 한 줄 요약

100V / 130A N-MOSFET — [[Brake Resistor Circuit]] 제동 스위치. 재생 전류를 3Ω 저항으로 흘림.

## 사양

- V_DSS: 100V
- I_D: 130A 연속
- R_DS(on): 2.5mΩ @ V_GS=10V
- 패키지: TO-263 (D2PAK, 방열 패드)

## 왜 100V / 130A

- V_DSS 100V: 재생 서지 (37.8V 클램프) + 마진 2×
- I_D 130A: 피크 10A 전류 흘릴 때 여유
- R_DS 2.5mΩ: 10A × 10A × 2.5mΩ = **0.25W** 손실 (발열 미미, 방열 쉬움)

## 연결

```
GND ── SOURCE
3Ω 저항 ── DRAIN
TLV3201 비교기 출력 ── 10Ω ── GATE
```
