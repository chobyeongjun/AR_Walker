# TPS62933

## 한 줄 요약

2.5 MHz 동기식 buck, 60V 입력 / 2A 출력 — 모터 라인에서 5V 로직 전원 분기.

## 왜 선택했나

- **60V 입력** → 12/24/36/48V 모두 커버 → Phase A 에서 토폴로지 freeze 가능
- **2.5 MHz 스위칭** → 인덕터·캡 소형 (vs TPS54360 400 kHz 대비 면적 ~30% ↓)
- 내장 MOSFET (외부 스위치 불필요)
- 스페이스 절약 (SOT583 2×2mm)

## 외부 회로

- 인덕터: **2.2 µH** (Coilcraft XAL4040-222 또는 동급, 4×4 mm)
- 입력 캡: 22µF/63V × 1 + 1µF/63V 세라믹 (Phase B 에서 V 정격 확정)
- 출력 캡: 22µF/16V × 2 + 100nF
- 피드백 저항 분압 (5V 출력 기준 — 데이터시트 공식)

## 연결되는 블록

- [[Logic Power]] — buck → LDO 로 이어짐

## 주의

- 입력 캡 V 정격은 [[Voltage-Current Scaling]] 매트릭스 참조 후 Phase B 에서 확정
- 입력단에 ferrite bead 배치해서 모터 노이즈 격리 ([[Motor Power Isolation]])
