# TCAN1462

## 한 줄 요약

CAN-FD 트랜시버 (5 Mbps) — Elmo 드라이버와 CANopen 통신.

## 사양

- CAN-FD 지원: 최대 5 Mbps
- 로직 레벨: 3.3V / 5V (VIO 핀)
- 전원: 5V (VCC)
- ESD 보호: ±8 kV IEC
- 대기 모드 지원

## 왜 선택했나

- [[Elmo 드라이버]] 는 CAN-FD / CANopen 표준 지원
- 5 Mbps 로 빠른 데이터 교환 (1kHz 제어 루프 여유)
- VIO 3.3V 분리 → MCU 직결 가능 (레벨 쉬프터 불필요)
- TI 제품 → ST HAL · 레퍼런스 풍부

## 외부 회로

- 120Ω 종단 저항 + 점퍼 (버스 끝단 선택)
- PESD2CAN ESD 다이오드
- CANH / CANL 공통모드 초크 (옵션)
- 5V 바이패스 100nF + 10µF
