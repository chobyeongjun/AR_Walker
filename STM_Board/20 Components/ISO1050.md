# ISO1050

> **확정 CAN 트랜시버** — 5000 Vrms 갈바닉 격리 내장. 내 원안 [[TCAN1462]] (비격리) 에서 변경.
> 변경 이유: 이전 보드에서 **GND bounce 2.5V 로 TXD/RXD (±0.3V 절대최대) 파손** → 격리 필수.

## 한 줄 요약

STM32 ↔ Elmo 드라이버 CAN 버스 간 **물리적 격리 경로**. MCU GND 와 모터 전원 PGND 가 전기적으로 끊어져 ground bounce · 모터 노이즈 · 서지가 MCU 로 전파되지 않음.

## 사양

| 항목 | 값 |
|---|---|
| 격리 내압 | **5000 Vrms** (1분 정격) |
| 데이터 레이트 | 1 Mbps (CAN 2.0), CAN-FD 지원 X |
| VCC1 (로직 측) | 3.3V (STM32 AGND 도메인) |
| VCC2 (버스 측) | 5V (PGND 도메인) |
| 로직 입력 | 3.3V / 5V 호환 |
| 절대최대 VCC2 | 6.5V |
| 동작 온도 | -55 ~ 105°C |
| 패키지 | SOIC-16 wide body |
| 가격 | ~$4 |

**⚠️ 제약:** CAN-FD **미지원** (클래식 CAN 만, 1 Mbps).
→ Elmo 드라이버가 CANopen over 클래식 CAN 이면 OK. FD 필요하면 ISO1042 (CAN-FD 2Mbps 격리) 검토.

## 왜 ISO1050 인가

→ [[Isolated CAN]] · [[GND Bounce Protection]]

### Legacy review 에서의 변경 추적

| 버전 | CAN 트랜시버 | 보호 |
|---|---|---|
| legacy 초기 | TJA1051T/3J (비격리) | SMAJ5.0A (클램프 9.2V) |
| legacy review 후 | **ISO1050 (격리)** | **BZT52C5V6 5.6V 제너** (VCC2 절대최대 6.5V 보호) |
| 내 원안 | TCAN1462 (비격리) | 일반 ESD |
| **최종** | **ISO1050 (격리)** | legacy review 와 동일 |

### 격리 안 하면 어떻게 되나 (이전 보드 사고 재현)

```
모터 리턴 전류 급변 50A/µs + PCB GND 인덕턴스 50nH
→ V_bounce = L × dI/dt = 2.5 V (두 GND 도메인 간)

CAN 트랜시버 (비격리) TXD/RXD 핀:
  절대최대정격: VIO ± 0.3V
  실제 인가: 2.5V >> 0.3V → 즉시 파손
```

ISO1050 은 **물리 경로 자체를 끊음** → V_bounce 가 전파되지 않음.

## 외부 회로

```
[MCU 측 3.3V AGND 도메인]
FDCAN1_TX (PD1) ─[33Ω]─ TXD
FDCAN1_RX (PD0) ─[33Ω]─ RXD
3.3V ─[100nF + 1µF]─ VCC1
GND(AGND) ─────────── GND1

        ══ 격리 경계 ══
        (SOIC-16 wide body, 6mm creepage)

[버스 측 5V PGND 도메인]
VCC2 ← [MGJ2D05050SC]  5V → 5V 2W 격리 DC/DC
GND2 = PGND
VCC2 ─[BZT52C5V6]─ GND2  (5.6V 클램프, 절대최대 6.5V 보호)

CANH ─[22Ω]─[CMC ACM2012-900]─[SM712-02HTG TVS]─ CAN_H 커넥터
CANL ─[22Ω]────────────┘                       ─ CAN_L 커넥터
R_TERM 120Ω (버스 종단)
```

## 격리 DC/DC (짝 부품)

- **MGJ2D05050SC** (Murata, 5V→5V 2W 격리) — VCC2 독립 공급
- 격리 내압 DC/DC 자체도 5000+ Vrms
- 작은 모듈 (SIP-7, ~$10)

## 배치 규칙 (PCB 레이아웃)

→ [[Motor Power Isolation]] 참고

- ISO1050 패키지가 AGND ↔ PGND 경계를 **수직으로 가로지름**
- Layer 1~N 모두 솔더마스크 갭 **≥2 mm** (creepage/clearance)
- ISO1050 아래 Layer 2 GND 플레인에도 갭 (via 없음)

## 커넥터에서 거리 (legacy 규칙)

```
       → 18mm : ISO1050 버스측

       → 28mm : 33Ω MCU 측
       → 35mm : STM32 FDCAN 핀
```

## CAN-FD 대안 (미래)

Elmo 드라이버가 CAN-FD 2Mbps 모드로 갈 경우:
- **ISO1042** (TI, 격리 CAN-FD, 5 Mbps) — ISO1050 의 FD 버전, 핀 호환
