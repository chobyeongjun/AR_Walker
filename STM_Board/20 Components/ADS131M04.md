# ADS131M04

## 한 줄 요약

24-bit ΔΣ ADC, 32 kSPS, 4채널 동시 샘플링 — 케이블 텐션 force feedback 1kHz 제어 루프용.

## 왜 선택했나

HX711 (80Hz max, 24-bit but slow, 1채널) → **ADS131M04** 로 강화 (사용자 요청 "앰프 기능 향상").

- **32 kSPS** — 1kHz 제어 루프 + 4× 오버샘플링 여유
- **4채널 동시 샘플** — 두 케이블 + 예비 2채널
- **simultaneous sampling** — force-pose 정렬에 중요 (phase error 없음)
- **SPI + DMA** — MCU 오버헤드 최소

## 핵심 연결

- [[MCU Core]] SPI1 + DRDY EXTI + SYNC_RESET GPIO
- [[REF5025]] 2.5V 정밀 레퍼런스 → REFIN
- 3.3V analog rail ([[Logic Power]] ferrite 분리) → AVDD
- 로드셀 4선 (E+, E-, S+, S-) × 2 → JST-GH 커넥터

## 주의사항

- AVDD/DVDD 별도 ferrite 분리 필수 (정밀도)
- REF 노이즈 감쇠 캡 (NR pin → 1µF) 필수
- 차동 입력에 RC 안티앨리어싱 (fc ≈ 16 kHz)
- Chopper-stabilized → 1/f noise 억제, DC 측정 우수
