# SL08-2R005

## 한 줄 요약

2Ω NTC 인러시 제한. 2개 직렬 (4Ω) 로 반복 동작 지원.

## 사양

- 저항 (25°C): 2Ω
- 최대 연속 전류: 5A
- 에너지 정격: 5J (단발)
- 패키지: through-hole

## 사용 방식

[[Inrush Current Limiting]] 참조.

2개 직렬로 4Ω → I_inrush = 24V/4Ω = 6A. τ = 45ms.
100ms 후 Omron G5V-1 릴레이 bypass → NTC 발열 없음.
