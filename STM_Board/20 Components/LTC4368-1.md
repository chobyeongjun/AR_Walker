# LTC4368-1

## 한 줄 요약

0~80V eFuse 컨트롤러 — OV/UV/RV/OCP/inrush 를 외부 N-MOSFET 1개로 통합 보호.

## 왜 선택했나

→ 자세한 이유는 [[eFuse]] 노트 참조

- 전압 범위 **0~80V** → 24/36/48V 모두 여유
- latch 모드 (-1 버전) → OVP/OCP 발생 시 명시적 알림 + reset 필요
- 외부 N-MOSFET 1개 ([[IPB180N06S4]]) 로 15A+ 통과
- POWER_GOOD 핀 → MCU EXTI 로 이벤트 알림

## 왜 TPS25940 이 아닌가

- TPS25940 은 4~18V — 우리 24V+ 에 불충분
- MAX17608 은 60V max — 48V 마진 빠듯

## 외부 회로 (핵심)

```
VIN ──[SENSE resistor]──┬── SOURCE_MOSFET  ──► VOUT
                        │
                        └── LTC4368 SENSE 입력
                             │
                             ├── OVP 저항 분압 (R1/R2)
                             ├── UVLO 저항 분압
                             ├── IMON 저항 (OCP 임계값)
                             ├── dV/dt 캡 (inrush ramp)
                             └── /FAULT → MCU EXTI
                                 /POWER_GOOD → MCU GPIO
```
