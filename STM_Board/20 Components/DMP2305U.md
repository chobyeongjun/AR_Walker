# DMP2305U-7

## 한 줄 요약

-30V / 4A P-channel MOSFET — 역접속 보호 (ideal diode).
[[BZT52C12]] 12V 제너와 함께 사용 (VGS 절대최대 ±20V 보호).

## 사양

- V_DSS: -30V
- I_D: -4A (연속)
- **V_GS 절대최대: ±20V** ⚠️
- R_DS(on): 78mΩ @ V_GS=-10V
- 패키지: SOT-23

## ⚠️ Legacy Review C3 (중요)

24V 배터리 역접속 시:
```
V_GS = -22.9V (배터리 완충 25.2V 에서)
V_GS 절대최대 ±20V 초과 → P-MOS 파손
```

**해결: GATE-GND 에 12V 제너 [[BZT52C12]] + R 100kΩ 풀업**

```
V_BATT (+) ───[R 100kΩ]── GATE ──[BZT52C12 12V]── GND
                             │
                           SOURCE
                             │
                           DRAIN ── Bus +
```

정상: GATE-GND = 12V (제너 클램프) → V_GS = -12V (정상 ON, R_DS 양호)
역접속: GATE 가 V_BATT(-) 방향 → MOSFET OFF → 역전류 차단
