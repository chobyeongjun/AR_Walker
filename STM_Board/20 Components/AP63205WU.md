# AP63205WU-7

> Legacy review C1 에서 선정. 내 원안 [[TPS62933]] 와 역할 동일 — 24~48V → 5V.

## 한 줄 요약

500kHz 동기식 버크, 32V 입력 / 2A 출력. 배터리 완충 25.2V 수용.

## Legacy 가 이걸 선택한 이유

초기 설계의 AP62200WU 가 VIN 최대 **24V** → 배터리 완충 **25.2V** 초과 → review 에서 AP63205WU-7 (VIN 32V) 로 교체 (C1).

## 내 TPS62933 과 비교

| 항목 | AP63205WU-7 (legacy) | [[TPS62933]] (내 추천) |
|---|---|---|
| Vin max | 32V | **60V** |
| Iout max | 2A | 2A |
| 스위칭 | 500 kHz | **2.5 MHz** |
| 인덕터 크기 | 큼 | 작음 |
| 48V 시스템 | ❌ | ✅ |
| Phase B 48V 옵션 대응 | 교체 필요 | 대응 가능 |

### 판단

- **24V 시스템 확정이면**: AP63205WU-7 (legacy 선택)
- **48V 옵션 살려야 하면**: TPS62933 (60V 커버)

사용자가 Phase B 에서 48V 갈 가능성 있으면 TPS62933 가 나음. 현재 미정이므로 **둘 다 후보로 유지**, 최종 전압 결정 후 확정.

## 현재 상태

Phase B 에서 배터리 전압 최종 확정 후 선정. 24V 확정이면 AP63205WU-7, 48V 면 TPS62933 유지.
