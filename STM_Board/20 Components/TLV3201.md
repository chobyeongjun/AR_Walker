# TLV3201

## 한 줄 요약

단전원 푸시-풀 비교기 — [[Brake Resistor Circuit]] 에서 26.5V 임계 감지 → [[IPB025N10N3]] MOSFET 게이트 구동.

## 사양

- 전원: 2.7~5.5V 단전원
- 공통 입력 범위: V- ~ V+ (rail-to-rail 입력)
- 응답 시간: 40ns (빠름)
- 출력: push-pull (MOSFET 직접 구동 가능)
- 패키지: SC-70-5 (1.6×1.6mm)

## 분압 회로

```
V_bus ─[R1: 100kΩ]─┬─[R2: 10kΩ]─ GND
                     │
                   TLV3201 IN+
                     │
      [Vref = 2.41V] ── TLV3201 IN-

임계 V_bus = 2.41 × (100k + 10k) / 10k = 26.5V
```
