# TLV75533

## 한 줄 요약

저노이즈 3.3V LDO, 500 mA — 5V buck 출력 → 디지털·아날로그 3.3V 생성.

## 왜 선택했나

- 저노이즈 (PSRR 70 dB @ 1kHz) — ADC 정밀도 유지
- SOT-23-5 (2.9×1.6mm) — 컴팩트
- 500 mA → MCU + 주변기기 전체 커버

## 2× 사용 (디지털 / 아날로그 분리)

1. **디지털 3V3** — MCU core, DVDD, GPIO, 통신 IC
2. **아날로그 3V3** — ferrite bead 경유 → [[ADS131M04]] AVDD, [[REF5025]] VIN, MCU VDDA/VREF+

→ 이유: [[Loadcell Amplifier Design]] SNR 향상

## 외부 회로

- 입력 캡: 1µF/10V 세라믹
- 출력 캡: 1µF/10V 세라믹 (X5R 이상)
- (옵션) NR 핀 캡 — 노이즈 감쇠
