# IPB180N06S4

## 한 줄 요약

60V / 180A N-channel MOSFET, RDS(on) 2.7 mΩ — [[LTC4368-1]] eFuse 의 통과 소자.

## 사양

- V_DS: 60V (24/36/48V 모두 커버)
- I_D: 180A 연속
- RDS(on): 2.7 mΩ @ V_GS=10V
- 게이트 전하: ~170 nC
- 패키지: TO-263 (D2PAK, 방열 패드)

## 왜 선택했나

- 60V 여유 (48V 입력 + 회생 surge 대비)
- 낮은 RDS(on) → 15A × 15A × 2.7mΩ = **0.6W 손실** (수용 가능)
- TO-263 방열 패드 → 폴리곤 연결로 열 분산
