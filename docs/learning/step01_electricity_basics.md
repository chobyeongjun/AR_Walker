# Step 1: 전기의 기본 3요소

> 전기의 모든 것은 **전압, 전류, 저항** 이 3가지에서 시작한다.

---

## 1. 전압 (Voltage, V)

**비유**: 물탱크의 **수압** (높이)

- 단위: **볼트 (V)**
- 전기를 밀어내는 "힘" 또는 "압력"
- 배터리의 +극과 -극 사이의 전위차

```
높은 곳의 물탱크 → 수압이 높음 → 물이 세게 흐름
48V 배터리      → 전압이 높음  → 전류를 세게 밀어냄
```

### AR Walker에서의 전압
| 위치 | 전압 | 용도 |
|------|------|------|
| 배터리 | **48V** | Elmo 모터 드라이버 전원 |
| 벅 컨버터 출력 | **5V** | 센서, 일부 IC 전원 |
| LDO 출력 | **3.3V** | STM32, ESP32 로직 전원 |

---

## 2. 전류 (Current, I)

**비유**: 파이프를 통해 흐르는 **물의 양**

- 단위: **암페어 (A)**, 작은 값은 밀리암페어 (mA) = 0.001A
- 실제로 회로를 통해 흐르는 전자의 양
- 전류가 많이 흐르면 → 발열 증가, 더 많은 일을 할 수 있음

```
파이프에 물이 많이 흐르면 → 수차(물레방아)가 빠르게 회전
회로에 전류가 많이 흐르면  → 모터가 강하게 회전
```

### AR Walker에서의 전류
| 부품 | 대략적인 전류 | 설명 |
|------|-------------|------|
| Elmo + 모터 | 수 A ~ 수십 A | 가장 큰 전력 소비자 |
| STM32H743 | ~100-300 mA | 제어 연산 |
| ESP32-S3 | ~100-300 mA | Wi-Fi/BLE 통신 시 |
| INA828 (로드셀 앰프) | ~1-5 mA | 매우 적은 전류 |
| Jetson Orin Nano | ~7-15W (5V에서 1.4~3A) | AI 연산 |

---

## 3. 저항 (Resistance, R)

**비유**: 파이프의 **좁기** (굵기의 반대)

- 단위: **옴 (Ω)**, 큰 값은 킬로옴 (kΩ) = 1000Ω
- 전류의 흐름을 방해하는 정도
- 저항이 크면 → 전류가 적게 흐름
- 저항이 작으면 → 전류가 많이 흐름

```
파이프가 좁으면  → 물이 적게 흐름
저항이 높으면    → 전류가 적게 흐름
```

### 저항은 어디에 쓰이나?
- **LED 보호**: LED에 직접 전압을 걸면 타버림. 저항으로 전류를 제한
- **풀업/풀다운**: 디지털 핀의 기본 상태를 정해줌
- **전압 분배**: 높은 전압을 낮춰서 ADC에 입력

---

## 4. 옴의 법칙 (Ohm's Law) ★★★

> **V = I × R**

이것만 외우면 전기의 절반은 끝이다.

```
V (전압) = I (전류) × R (저항)
I (전류) = V (전압) / R (저항)
R (저항) = V (전압) / I (전류)
```

### 삼각형으로 외우기
```
    [V]
   /   \
 [I] × [R]

- V를 구하려면: I × R
- I를 구하려면: V / R
- R를 구하려면: V / I
```

---

## 5. 전력 (Power, P)

> **P = V × I** (와트, W)

- 단위: **와트 (W)**
- 전기가 하는 "일의 양" 또는 "발열량"
- 전력이 높으면 → 더 많은 일을 하거나 더 뜨거워짐

### AR Walker 전력 계산 예시
```
Elmo 모터: 48V × 10A = 480W (최대 부하 시)
STM32:     3.3V × 0.2A = 0.66W
ESP32:     3.3V × 0.3A = 0.99W
Jetson:    5V × 3A = 15W
```

**왜 48V를 쓰는가?**
```
같은 480W 전력을 전달할 때:
  24V 시스템: 480W / 24V = 20A 필요 → 전선 굵어야 함, 발열 큼
  48V 시스템: 480W / 48V = 10A 필요 → 전선 절반, 발열 절반
```
→ 48V는 같은 파워를 **절반의 전류**로 전달 = 효율적!

---

## 6. 직렬 vs 병렬 회로

### 직렬 (Series)
```
배터리 ──[R1]──[R2]──[R3]── 배터리(-)
```
- 전류: 모든 부품에 **같은 전류**가 흐름
- 전압: 각 부품에서 **나눠짐** (V1 + V2 + V3 = V총)
- 저항: R총 = R1 + R2 + R3

### 병렬 (Parallel)
```
         ┌──[R1]──┐
배터리 ──┤──[R2]──├── 배터리(-)
         └──[R3]──┘
```
- 전류: 각 가지로 **나눠짐** (I1 + I2 + I3 = I총)
- 전압: 모든 부품에 **같은 전압**
- 저항: 1/R총 = 1/R1 + 1/R2 + 1/R3

### AR Walker에서는?
- 모터, MCU, 센서 등은 모두 전원에 **병렬**로 연결됨
- 각 부품이 필요한 만큼의 전류를 독립적으로 가져감

## 연습 문제 (직접 풀어보기!)

### 문제 1: 기본 옴의 법칙
> LED에 3.3V를 걸고, 20mA를 흘리고 싶다.
> LED 자체 전압강하가 2V라면, 필요한 저항은?

<details>
<summary>정답 보기</summary>

```
저항에 걸리는 전압 = 3.3V - 2V = 1.3V
필요한 저항 = V / I = 1.3V / 0.02A = 65Ω
→ 가장 가까운 표준 저항값: 68Ω 사용
```
</details>

### 문제 2: 전력 계산
> 48V 배터리에서 Elmo 모터 드라이버가 최대 15A를 소비한다.
> 최대 전력은 얼마인가?

<details>
<summary>정답 보기</summary>

```
P = V × I = 48V × 15A = 720W
```
</details>

### 문제 3: 왜 48V인가?
> 720W를 24V로 공급하면 전류가 얼마나 필요한가?
> 48V와 비교하면?

<details>
<summary>정답 보기</summary>

```
24V: I = P / V = 720W / 24V = 30A
48V: I = P / V = 720W / 48V = 15A

48V는 24V 대비 전류가 절반!
→ 전선 단면적 절반, 발열 1/4 (P_loss = I²R이므로)
```
</details>

| 개념 | 기호 | 단위 | 비유 | AR Walker 예시 |
|------|------|------|------|---------------|
| 전압 | V | 볼트 (V) | 수압 | 48V 배터리 |
| 전류 | I | 암페어 (A) | 물의 양 | 모터 15A |
| 저항 | R | 옴 (Ω) | 파이프 좁기 | LED 보호 저항 |
| 전력 | P | 와트 (W) | 일의 양 | 720W 모터 |

**공식 2개만 외우자:**
1. **V = I × R** (옴의 법칙)
2. **P = V × I** (전력)
